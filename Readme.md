# AI-Based Diabetes Risk Predictor

An AI-powered full-stack web application that predicts a user's risk of developing diabetes using Machine Learning.

The system analyzes health parameters such as glucose level, BMI, age, blood pressure, and pregnancy count to calculate a risk probability and classify it as Low, Medium, or High risk.
